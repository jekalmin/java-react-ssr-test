# java-react-ssr-test
## Objective
- Test Server Side Rendering(SSR) of React using Spring Boot
- Performance difference between nashorn and j2v8

## Build and Run
```sh
# run webpack
npm run build --prefix src/main/resources
```
```sh
# tomcat start
mvn spring-boot:run
```
Entry URL: 

http://localhost:8080/test
